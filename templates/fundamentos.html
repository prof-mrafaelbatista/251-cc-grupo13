Fundamentos da programação em Python

Estruturas de seleção

As estruturas de seleção, também conhecidas como estruturas condicionais, são utilizadas para controlar o fluxo de execução do código dependendo de uma condição. Elas são essenciais para tomar decisões dentro de um programa. As estruturas mais comuns são if, elif e else.

if (Se): A estrutura if avalia uma condição. Se a condição for verdadeira, o bloco de código dentro da estrutura if é executado. Caso a condição seja falsa, o bloco é ignorado, e o código continua a partir da próxima instrução.

else (Senão): A estrutura else é utilizada em conjunto com a estrutura if. Se a condição do if for falsa, o bloco de código dentro da estrutura else é executado.

elif (Senão Se): A estrutura elif permite adicionar condições adicionais para verificar se uma condição anterior foi falsa e, em seguida, executar um bloco de código diferente se a condição elif for verdadeira.

Estruturas de repetição

As estruturas de repetição são utilizadas para executar um bloco de código várias vezes. As principais estruturas de repetição em Python são for e while. O for é usado quando se sabe o número exato de repetições ou quando se precisa iterar sobre um iterável, como uma lista ou string. O while é usado quando a repetição depende de uma condição lógica, que deve ser verdadeira para o loop continuar.

Na estrutura de repetição for sua sintaxe vem representada da seguinte forma: "for variável_de_controle in objeto_iterável:", onde a variável_de_controle recebe o valor de cada elemento do objeto iterável; e o objeto_iterável pode ser uma lista, tupla, string, conjunto, etc.

Na estrutura de repetição while sua sintaxe vem representada da seguinte forma: "while condição:", onde a condição é uma expressão lógica que determina se o loop continua.

Vetores e Matrizes

Vetores (ou listas) são estruturas que armazenam uma sequência de elementos. Matrizes são listas de listas, representando dados em duas dimensões. Essas estruturas são fundamentais para armazenar e manipular coleções de dados, como séries temporais, tabelas ou grades de pixels.

Funções e Procedimentos

Funções e procedimentos são blocos de código reutilizáveis que executam tarefas específicas. A principal diferença reside no retorno de valor: as funções retornam um valor, enquanto os procedimentos não.

Uma função é um bloco de código que realiza uma tarefa específica e pode receber argumentos como entrada e retornar um valor como saída. As funções são projetadas para retornar um valor, o que permite que o código que as chama possa utilizar o resultado da execução.

Um procedimento é um bloco de código que executa uma tarefa, mas não é projetado para retornar um valor. Procedimentos não retornam valores, ou seja, a sua execução pode ter efeitos colaterais (como imprimir algo na tela ou modificar dados), mas não retornam um valor que pode ser utilizado pelo código que os chama.

Tramento de exceção

O tratamento de exceções permite que o código lide com erros de forma controlada, em vez de interromper a execução do programa. A estrutura básica para tratamento de exceções é try...except, que encapsula o código que pode lançar uma exceção e, em seguida, captura e trata a exceção se ocorrer.

O "try" contém o código que pode lançar uma exceção. Se uma exceção ocorrer durante a execução deste bloco, a execução é interrompida e o controle passa para a cláusula except correspondente.

O "except" contém o código a ser executado se uma exceção específica for lançada no bloco try. Você pode ter várias cláusulas except para lidar com diferentes tipos de exceções.

Por fim, o "except Exception as e:" esta cláusula captura todas as exceções que não forem especificamente tratadas por outras cláusulas except. e é uma variável que armazena a instância da exceção capturada.
